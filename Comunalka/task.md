# Облік ресурсів

```
База даних

Ресурс (електрика, холодна вода, газ)
	ід
	назва

Тариф
	ід
	ресурс
	вартість за одиницю

Лічильник
	ід
	ресурс
	тариф
	номер


Показ лічильника
	ід
	лічильник
	дата зняття показу
	значення


Запити до бази даних повинні працювати асинхронно

Графічний інтерфейс на WPF
	таблиця ресурсів (додавати, редагувати, видаляти)
	таблиця тарифів (додавати, дерагувати)
	таблиця лічильників (додавати, редагувати, видалити, додати показ лічильника)
	показувати останній показ, та скільки за місяць витрачено ресурсів
	таблиця показів лічильника, та вартість витраченого ресурсу

```
